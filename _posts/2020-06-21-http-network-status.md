# 2W1H

## Q1. 상태 코드란 무엇인가?

클라이언트가 보낸 리퀘스트가 성공했는지 실패했는지 서버가 알려주는 코드이다.

## Q2. 상태 코드 클래스 5가지를 일련의 흐름으로 외우는 방법을 설명하라.

클라이언트가 서버에 리퀘스트를 보내는 일련의 흐름을 순서대로 진행하되, 성공인 경우를 먼저 고려한다.

1xx: 처리중 > 2xx: 처리 완료 > 3xx: 처리하려면 추가 동작 필요 > 4xx: 클라이언트가 보냈을 때 에러 > 5xx: 서버에서 처리할 때 에러

- 1xx: informational 리퀘스트 받아서 처리중
- 2xx: success 정상 처리 완료
- 3xx: redirection 완료하려면 추가 동작 필요
- 4xx: client error 클라이언트가 보낸 리퀘스트가 잘못되어 에러
- 5xx: server error 서버 측 문제로 에러
