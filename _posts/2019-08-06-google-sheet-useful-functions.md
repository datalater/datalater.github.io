# 목적

구글 스프레드시트를 자동화 도구로 사용하면서 문제를 겪을 때마다 유용하게 사용했던 함수들을 지속적으로 기록한다.

# 문제 상황 및 해결책


## 셀 번호가 적힌 문자열을 참조하고 싶을 때

예외사항이 발생한다면, 템플릿의 A15 셀에 안내메시지를 출력해야 한다. 그리고 예외사항 여부는 A19 셀의 텍스트 내용으로 판별할 수 있다.

만약 예외사항이 발생하지 않을 경우, 템플릿에 있는 19행 자체가 삭제되고 그 아래에 있던 20행의 내용이 19행으로 당겨진다.

결국 A19 셀의 값이 예외사항 여부를 판별할 수 있는 기준이 되므로 A15 셀에 다음과 같은 함수를 작성했다.

```
=if(A19 = "예외사항", "* 예외사항이 적용됩니다.", "")
```

문제는 예외사항이 발생하지 않을 경우, 19행 자체가 삭제되고 20행에 있던 내용이 19행으로 당겨지는 과정에서 19행 자체가 삭제될 때 위 함수가 `#REF`라는 참조 오류를 출력하면서 작동하지 않는다는 점이다. 참조할 행이 삭제되어 참조할 수 없다는 오류다. 로직을 그대로 사용하고 싶다면 다음과 같이 하면 된다:

```
=if(indirect("A19") = "예외사항", "* 예외사항이 적용됩니다.", "")
```
