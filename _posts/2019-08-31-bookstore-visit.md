# 소감

AWS는 아직 잘 모르기 때문에 제목만 봐도 손이 간다. [AWS 기반 서버리스 아키텍처](http://www.yes24.com/Product/goods/57838923)는 앞에 20 페이지 정도를 훑어보았는데, 인프라를 신경쓰지 않고 코드만 신경쓰게 하려는 하나의 흐름이라는 말이 인상깊었다. 내게 주어진 일 중에서 아키텍처로 AWS lambda를 고려하는 업무가 있기 때문에 조만간 알아보게 될 것 같다.

[누구나 쉽게 배우는 클로저](http://www.kyobobook.co.kr/product/detailViewKor.laf?ejkGb=KOR&barcode=9791186697221)는 아샬님이 라이브 방송 중에 클로저를 주언어로 쓰도록 하겠다는 한 마디가 꽂혀서 꺼내본 책이다. 함수형 언어를 익히는데 기반이 될 것 같은 느낌이 들었다.

익스프레스JS를 사용해야 하는 업무가 있어서 [자바스크립트로 서버와 클라이언트 구축하기](http://www.yes24.com/Product/Goods/61848347)를 꺼내보았다. 목차를 보니 입문자에게 친절히 알려주는 구성으로 느껴졌다. 다만, 국내에서 출판되는 책 중 간간이 보이는 부자연스러운 컬러톤(이 책에서는 분홍색) 때문에 가독성이 떨어져 보였고, 표지에 적힌 소스코드 링크가 github이 아니라 출판사 웹 사이트라는 것에서도 호감도가 떨어졌다.
